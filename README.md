# Mastan Portfolio

A modern, responsive portfolio website built with React and Vite.

## Features
- **Custom Animations**: Smooth transitions and effects using CSS and Canvas.
- **Glassmorphism UI**: Premium visual style matching the "Rich Aesthetics" requirement.
- **Responsive Design**: Adapts to all screen sizes.
- **Canvas Particles**: Interactive background.

## Directory Structure
Follows the requested structure:
- `src/components/{animated, sections, ui}`
- `src/{hooks, utils, styles}`
- `public/assets/{images, fonts, icons}`

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
